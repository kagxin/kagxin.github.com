mysql trigger触发器使用

2017-07-19

创建触发器的一般语法

before 和 after参数指定了触发器执行时间，
触发事件 insert、update、delete
for each row 表示任何一条记录上的操作满足触发事件都会触发该触发器。

mysql中， 一个表在相同触发时间的相同触发事件，只能创建一个触发器
example: department 表中，触发事件insert，触发时间为after的触发器只能有一个。

new old

对于INSERT语句,只有NEW是合法的；对于DELETE语句，只有OLD才合法；而UPDATE语句可以在和NEW以及OLD同时使用。
